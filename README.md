# Grund-Idee
Meine App soll den User dabei helfen sich Gewohnheiten anzueignen und sie zu behalten. Die beste Übersicht über Gewohnheiten, die man sich aneignen will, hat man, wenn man sich jeden Tag notiert, ob man die Gewohnheit heute gemacht hat und gegebenenfalls, wie nah man an dem Ziel gekommen ist. Somit sieht der User über einen Kalender wie oft er seine Vorhaben umgesetzt und hat somit ein realistisches Feedback. Jede Woche wird ein Rückblick erstellt wieviel Prozent der Vorhaben man in dieser Woche umgesetzt hat. Wenn die Prozentzahl zu niedrig ist, weiss der User, dass er lieber an wenigeren Gewohnheiten arbeiten soll, diese aber dafür auch wirklich regelmässig macht damit sie zur Gewohnheit werden. Wenn man eine Gewohnheit hat, wie 10’000 Schritte pro Tag machen, werden die Schritte Daten direkt vom Handy gelesen und eingetragen. Gewohnheiten wie, etwas eine bestimmte Dauer lang tun, kann über ein Timer in der App gemessen werden. Dies wird dann auch direkt eingetragen. Gewohnheiten, die über den ganzen Tag verteilt sind, wie 3L Wasser pro Tag trinken, können immer zwischendurch eingetragen werden und es wird dann die Summe der einzelnen Einträge gezählt. Der User kann sich benachrichtigen lassen, um eine Aufgabe nicht zu vergessen.
